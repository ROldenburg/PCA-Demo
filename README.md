# PCA Demo

Diese Seite zeigt Python-Code, der eine Hauptkomponentenanalyse als Minimierungsproblem löst. Dadurch wir die Idee des Verfahrens deutlicher als bei einer Behandlung über lineare Alegbra, die die Optimerung implizit vornimmt.

Als Ergänzung gibt es auch eine Demo der k-Means-Clusterung, die oft in Verbindung mit PCA benutzt wird.
